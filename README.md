## Installation

### Method 1: From Source
1. Clone this repository
2. Run `npm install`
3. Run `npm run compile`
4. Copy the entire folder to your VS Code extensions directory:
   - Windows: `%USERPROFILE%\.vscode\extensions\markdown-preview-plus\`
   - macOS/Linux: `~/.vscode/extensions/markdown-preview-plus/`
5. Restart VS Code

## Usage

### Opening Preview
- **Keyboard Shortcut**: `Ctrl+Shift+M` (Windows/Linux) or `Cmd+Shift+M` (Mac)
- **Command Palette**: `Ctrl+Shift+P` ‚Üí "Show Markdown Preview"

### Opening Help
- **Keyboard Shortcut**: `Ctrl+Shift+H` (Windows/Linux) or `Cmd+Shift+H` (Mac)
- **Command Palette**: `Ctrl+Shift+P` ‚Üí "Show Markdown Help"
